* 데이터 처리 주기

개요
    기본 요청 주기: 1분 
    데이터 종류와 시간에 따라 갱신 조건 다르게 설정
    백그라운드 요청 중단 - 브라우저를 내리거나, 다른 창을 보고 있는 경우 갱신 중단

1. 개별 종목 시세 데이터
    주기: 1 분마다
    작동 시간: 08:00 ~ 20:00 (장전 시간외 + 정규장 + 장후 시간외/NXT)
    내용: 최신 가격, 거래량 수신

2. 투자자별 매매동향 차트 데이터
    기본 주기: 1 분마다 (차단 조건 시 제외)
    데이터: 실시간 (1~2 분 내외)
    시각화: 10 분 단위 슬롯 (09:10, 09:20, ...) 중 최신 값 덮어쓰기
    통신 차단 조건
       -주말	토, 일	API 요청 중단, 캐시만 표시
       -장 전	00:00 ~ 08:59	API 요청 중단, 어제 마감 캐시 표시
       -장 마감 후	15:30 이후	영구 캐시 모드 (자정까지 요청 중단)

3. 실시간 관련 뉴스 데이터
    주기: 1 분마다
    작동: 개별 종목 시세 갱신 성공 시 즉시 뉴스 요청
    내용: 관심 종목 최근 뉴스 1 분마다 새로 조회
    렌더링: 방금 올라온 뉴스 (방금, X 분 전) 표시
